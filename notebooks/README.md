# Jupyter Notebooks

This directory contains interactive notebooks for:

1. **01_data_exploration.ipynb**: Initial data exploration and understanding
2. **02_etl_pipeline.ipynb**: ETL pipeline development and testing
3. **03_postgres_analysis.ipynb**: PostgreSQL query analysis and optimization
4. **04_mongodb_analysis.ipynb**: MongoDB aggregation pipeline analysis
5. **05_comparative_analysis.ipynb**: Side-by-side comparison of both databases
6. **06_visualizations.ipynb**: Create charts and dashboards

## Usage

Start Jupyter:

```bash
jupyter notebook
```

Or use JupyterLab:

```bash
jupyter lab
```
